# Site Access Control

Site access control grants or revokes access in real time based on each worker's
live competency and induction status.

> Sample content for demonstration purposes.

## How access is determined

A worker is granted access only when **all** of the following are true:

- Their company is [pre-qualified](prequalify-a-company.md).
- Their required [competencies are verified](worker-competency.md).
- Their site induction is complete and current.
- No active suspensions or alerts apply.

## Managing access

1. Go to **Site → Access control**.
2. View workers currently on site and their access status.
3. Suspend or restore access for an individual when needed.
4. Review the access log for a full audit trail.

## Hardware integration

When connected to turnstiles, gates, or card readers, P2W 20:20 enforces these
rules physically — denying entry to anyone who doesn't meet the criteria.
