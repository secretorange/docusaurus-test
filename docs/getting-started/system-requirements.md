# System Requirements

P2W 20:20 is fully cloud-based — there is nothing to install on your servers.

> Sample content for demonstration purposes.

## Supported browsers

| Browser | Minimum version |
| --- | --- |
| Google Chrome | Last 3 releases |
| Microsoft Edge | Last 3 releases |
| Mozilla Firefox | Last 3 releases |
| Apple Safari | Last 2 releases |

## Network

- Outbound HTTPS (port 443) access to `*.gemsoft7.com`.
- A stable connection of at least 5 Mbps is recommended for site kiosks.

## Site access hardware (optional)

For real-time site access control, P2W 20:20 integrates with:

- Turnstiles and access gates
- RFID / proximity card readers
- Self-service induction kiosks

## Accounts and permissions

Access is role-based. Your administrator assigns each user one or more roles that
determine which assets and actions are available. See your administrator to
request a role change.
